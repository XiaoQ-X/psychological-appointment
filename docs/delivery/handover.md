# 安心心理三端系统交付总说明

生成时间：2026-06-01

## 交付范围

本次交付的是“高校学生心理健康预约服务平台”三端联动系统，包含学生端、咨询师端、管理员后台、统一后端 API、Prisma/MySQL 数据库和联动测试脚本。

| 端 | 地址 | 当前状态 |
|---|---|---|
| 学生端 | `http://localhost:5173` | 已完成 UI 原型复原、核心预约、测评、资讯、活动、消息、个人中心与 API 接入。 |
| 咨询师端 | `http://localhost:5174` | 已完成移动端小程序风格工作台、预约处理、学生档案、排班、风险、转介、资讯、消息与 API 接入。 |
| 管理员后台 | `http://localhost:5175` | 已完成 PC 后台风格看板、学生、咨询师、预约、排班、调班、风险、转介、文章、活动、校区、日志、设置与 API 接入。 |
| 后端 API | `http://localhost:3000` | 已完成核心 REST API、JWT 登录、业务闭环测试。 |
| 数据库 | `localhost:3307` | MySQL 8.0 Docker 容器，使用 Prisma 管理 schema 和 seed 数据。 |

## 技术栈

| 层级 | 技术 |

| 数据库 | MySQL 8.x，Docker Compose 默认映射到本机 `3307` |
| 认证 | JWT、bcryptjs |
| 工程 | npm workspaces |
